Esse site é resultado de uma pesquisa desenvolvida pela Escola de Engenharia da UFMG em parceria com a HEMOSC a fim de mostrar os dados da análise sobre a cadeia de suprimento de sangue no Brasil.

O site oferece recursos para fácil visualização dos dados que são utilizados como base para entender o panorama da distribuição das bolsas de sangue pelos hemocentros de Santa Catarina e orientar esse processo de forma mais eficiente e rápida.
